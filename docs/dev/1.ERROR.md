# Erros no primeiro projeto

Problemas encontrados no projeto anterior que dificultaram o desenvolvimento e que espero melhorar.

## do Project Owner/Scrum Master**

- Um template.html guia não foi criado anteriormente, então os "programadores" ficam sem base do layout do projeto
- Não foi definido um formato de indentação do código e/ou não foram utilizados ferramentas para

## do Grupo

- A documentação não foi alterada conforme o andamento do projeto
- Sprints entregues com varias pendências (Bugs, features pendentes ou razas, ...)
- Modelo do armazenamento de dados alterado constantemente
- WireFrames mal desenhados

## dos Indivíduos

- Dificuldades com o git
- Dependência de agentes de IA para gerar código

## O que é necessário melhorar?

- Documentação do projeto
- Scrum Mastering
- Reporte e solução de Bugs
- Organização das pastas/projeto e formatação
