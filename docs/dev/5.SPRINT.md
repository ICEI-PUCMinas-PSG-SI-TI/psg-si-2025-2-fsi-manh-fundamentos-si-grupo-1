## Recursos Não Funcionais

- O sistema deve validar tipos de dados e credenciais
- O sistema deve ter backend e camada lógica não deve ficar na aplicação web
- O sistema deve ter um design responsivo e deve se adaptar a dispositivos móveis
- O sistema deve manter logs de acessos via API
