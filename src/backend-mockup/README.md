# Mockup do Backend

Este diretório contem os arquivos do mockup do backend.

O mockup do backend foi realizado para que o desenvolvimento do frontend não seja dependente de como está o backend.

## Intruções

1 - Instalar as dependências do projeto

```sh
npm install
```

2 - Gerar arquivo mockup.json.

> O arquivo mockup.json irá conter varias informações geradas aleatoriamente de acordo com a estrutura de dados da aplicação.

```sh
npm run gen
```

3 - Rodar servidor de dados mockup

```sh
npm run serve
```
